有关hugo博客搭建和编辑的使用教程

Dec 30, 2021
2 min read

自从开始学习编程以后，我便将我所学全部都记录于笔记本和网络笔记，随着积累，我便有了想要搭建博客的想法
希望能将自己的学习历程和经验都分享给更多人，但万事开头难，我也尝试过很多次搭建，最终都以失败告终，查
找了很多资料，最终整理出了这份搭建方法，希望能对面前的你有所帮助，如果还是无法解决，也可以联系我的邮
箱，我会经我所能帮你解答疑惑。

首先在开始搭建前，需要准备以下几款软件
git
hugo
Vscode（这个软件是用来打开和编辑源代码的，有了会更方便一些，当然你用记事本也可以）

还需要一个github的账号，在官网 https://github.com 里注册就好，如果进不去那是因为DNS的污染，也是老生常谈的问题了，百度搜索一下有很多教程可以解决的，就不多赘述了

git下载地址：https://git-scm.com/
hugo下载地址：https://gohugo.io/getting-started/installing/ or 去hugo的官方github仓库 https://github.com/gohugoio/hugo/releases 下载对应的操作系统版本的Hugo二进制文件,如果你是64位系统就是hugo_0.55.1_Windows-64bit.zip，32位则对应另一个

下载完后需要配置环境变量，可以去看这个教程，很详细 https://baijiahao.baidu.com/s?id=1630800288039248525&wfr=spider&for=pc

在配置环境变量有一个坑
如果你是win8系统，点击path环境变量的编辑后他只会出现一个小框，没办法新建变量
而我最后的解决办法是, 首先在系统变量下面, 新建一个变量 变量名字是hugo 变量值写入你的下载Hugo的路径 我的路径是D：\Blog 直接填入然后确定 然后在系统变量的path里点击编辑 在变量值最后面用分号隔开填入你的hugo路径就可以 例子：
C:\Program Files\dotnet\;D:\Blog\;C:\Program Files\Git\cmd 中间就是hugo的变量

完成后环境的搭建就算大功告成了

这时候使用cmd来测试一下你的git 和 hugo 有没有装好 输入代码

git –help

如果出现长串说明代表成功 hugo同样

hugo –help

这里还有一个坑，一定要使用右键管理员启动cmd，不然有时候明明输入的代码正确却显示错误

一切都正常后开始下一步，因为我是windows，所以接下来会更偏向这个的使用习惯

首先在C盘外的其他盘建立一个文件夹，这里我用的是D盘，建立一个Blog文件夹，中文也就是博客 然后cmd命令
D：
cd blog
进入到blog文件夹里
然后使用Hugo指令创建一个本地博客
hugo new site myblog
成功后你可以发现在D：/blog/Myblog 里多了很多文件，可以点击打开查看，也可以直接输入
cd myblog
dir
来查看

这一步也成功后，接下来给你的博客选择一个好看的风格，也就是主题，进入到https://www.gohugo.org/theme/来选择 选好后按照你那个主页的链接进行下载
仔细观察可以发现在Installation下面有下载用的代码
这个代码其实无非就是，让你先创建一个名为themes的文件夹，在你博客的根目录下，然后进入到themes这个文件里，然后通过git clone (hugo的主题) 命令来下载到themes里
这里我直接用最简洁的办法
首先
cd [path]
就是先用cd命令到你的博客根目录下
git clone https://github.com/hugo-theme-m10c.git themes/m10c
然后使用git命令下载当前网址的主题到新建的 themes文件夹里的m10c文件夹里

成功下载以后就可以在 D:/blog/Myblog/themes/m10c/ 里看到又多了一堆文件，这些文件就是你刚下载的主题

如果无法下载，可能有两个原因，第一个原因是你的网络卡动，第二个原因是没有右键管理员运行cmd

成功后回到博客根目录下
然后输入
hugo server -t m10c –buildDrafts
来在本地启动你的博客
成功后会显示

如果显示这样，那就代表成功了 在最下排会有一个Web Server is available at http://localhost:1313/ (bind address 127.0.0.1)
将这个网址复制到浏览器里就能浏览本地博客了
成功后按Ctrl+C来结束运行

接下来让我们通过hugo命令来在本地发布第一个博客内容 首先回到博客的根目录
然后输入
hugo new post/blog.md
这是命令是新建一个Hugo博客内容 在根目录创建一个Post文件夹然后在里面创建一个名为blog.md的博客内容
成功后通过命令或者是手动去检查是否创建成功
成功后，接下来就是在这个Blog.md文件里去写你的博客内容，这时候就要用到我之前说的Vscode软件来打开，进去之后有三行代码，我们另起一行去写我们的内容就好了
这其中有一个标题的内容可以改，你可以在title这个属性的引号里去改一个标题

本地博客搭建好后，接下来就是将他推送到github的仓库里，让我们可以在互联网上浏览到

至于github的账号中注册和仓库创建，可以去网上百度教程，也是都能搜到，这里我推荐一个 https://www.bilibili.com/video/BV1tM4y1T7co?from=search&seid=6689601655732001844&spm_id_from=333.337.0.0

需要注意的地方是，注册github的名字需要是小些字母和数字，这样以后方便盯正，最好不要用符号，而创建仓库的时候，你的仓库地址要和你的名字是一样的，并且是以github.io为结尾

准备好后，接下来就是将本地博客部署到github的仓库里
首先回到博客的根目录，这个很重要，如果你的命令老是失败，很有可能就是没有回到根目录，所以一定要时刻查看
输入
bugo - - theme=m10c - - baseUrl=“https://blackcat-love.github.io/" - - buildDrafts （注意是双横杠，中间没有间隔）
这其中你注册时的名字填到里面就可以，成功后也显示一个方格子
这时候会生成一个public文件夹，这个文件里的东西就是要稍后发送到你的github仓库里的东西。

cd public
git init
git add .
git commit -m “我的Hugo博客第一次上传”

通过这四个命令将文件加到仓库里

git remote add origin https://github.com/你的github账号名字/你的github仓库地址.git
举例
git remote add origin https://github.com/blackcat110/balckcat110.github.io.git
这一部用来将你的博客和你的仓库关联

git push -u origin master
然后输入你的github账号和密码
注意，输入你的密码的时候，会自动隐藏，继续输入就好
成功后你的内容就会被推到仓库里，然后通过仓库的地址就可以在互联网上打开里

到这里就算是结束了，希望你可以成功搭建你的博客：）
